> 如果你的 Win11 电脑最近开始**打开软件慢、切窗口卡、开几个浏览器标签就死机**——多半不是硬件不行，是有一组后台服务在默默吃资源。下面这 5 个是关掉后**改善最明显**的，按影响从大到小排。

Win11 默认开启的很多功能，是为 2017 年那批 4GB 内存的笔记本设计的；在 2026 年的 NVMe + 16GB 机器上基本都是包袱。下面这 5 项是我最常看到"关完立竿见影"的设置。

## 5 个核心功能

| 功能 | 为什么关 | 怎么关 | 关完会怎样 |
|---|---|---|---|
| **SysMain**（超级预读 / Superfetch）| 预读"你可能要用"的程序——在机械硬盘上有用，在 SSD 上纯属烧内存和磁盘。还常常是"空闲时硬盘狂转"的元凶。| `Win+R` → `services.msc` → 找到 **SysMain** → 右键属性 → 启动类型 → **禁用** → 重启。| 8GB 内存机器上内存占用降约 25%，磁盘占用回到 10% 以下。|
| **开机自启项**（登录时自动启动的程序）| 每个都常驻后台，长期占一份 CPU + 一份内存。绝大多数不是你主动要的——是装软件时**被塞进来的**。| 任务管理器（`Ctrl+Shift+Esc`）→ **启动应用** 选项卡 → 不需要的右键 → **禁用**。**保守一点**：保留杀软、输入法、云盘同步这几个。| 开机速度 +40%，空闲 CPU 明显下降。|
| **传递优化**（Windows Update P2P 上传）| 用你的带宽和磁盘，**给互联网上其他 Win11 电脑**传更新碎片。除非你主动加入，否则你拿不到任何回报。| 设置 → **Windows 更新** → 高级选项 → **传递优化** → 关闭"允许从其他电脑下载"。| 磁盘占用稳到 15% 以下，上行带宽完全释放给你。|
| **视觉效果与动画**（阴影 / 透明 / 动画过渡）| 阴影、亚克力、平滑滚动、动画过渡——核显（或 5 年以上的老独显）上**真的吃帧率**，老软件里尤其明显。| `sysdm.cpl` → 高级 → 性能 → 设置 → **调整为最佳性能**。（想折中选"自定义" + 保留"平滑屏幕字体边缘"）| 切窗口明显跟手，弱 GPU 机器上 +20% 响应速度。|
| **后台应用**（你没在用但还在运行的程序）| 邮件、天气、新闻、OEM 装的小工具……都常驻后台刷新和发通知。每个都是一点点持续开销。| 设置 → **隐私和安全** → **后台应用** → 关掉你**不需要通知**的那些。| 空闲内存降 15–20%，笔记本续航明显改善。|

以上这些**都不是破坏性的**——最坏情况是改完感觉不习惯，再打开即可。如果你电脑是公司/学校 IT 管的设备，**部分开关可能被 IT 锁了**（灰色不可点），那对应的组策略其实已经帮你做了正确的设置，不用动。

## 顺手的 3 个辅助优化

1. **电源计划改成"最佳性能"** → 设置 → 系统 → 电源和电池 → 选**最佳性能**。笔记本**拔电**时切回"平衡"省电，**接电**时用"最佳性能"。这会解除 Win 默认的 CPU 频率节能上限。
2. **打开存储感知** → 设置 → 系统 → 存储 → 存储感知 → 开。设成**每周运行**。自动清临时文件、30 天后清回收站，保持 C 盘不爆满——C 盘满了**会悄悄让一切变慢**。
3. **跑一次 Windows 更新** → 设置 → Windows 更新 → 检查更新。微软在累积更新里发过**内存调度和磁盘 IO 修复**，没打补丁的 Win11 比打过补丁的**实测慢**。

## 改完自查 3 步

1. **重启**——SysMain 必须重启才生效；其他 4 项即时生效，但**干净的 reboot 才是测量的正确基线**。
2. **跑你日常用的程序**：浏览器十几个标签、IDE、Office 文档。光标卡不卡？任务管理器空闲 CPU 是不是在 5% 以下？
3. **每月维护一次**：手动跑一次存储感知；扫一遍启动项（每次装新 app 可能塞一个）；对**新加的后台应用**重复一遍"用不到就关"。

5 个核心 + 3 个辅助做完，**典型 8GB/i5/2018 笔记本**机器 1 小时内能感到差异——**空闲内存降 30-40%，硬盘不再空闲狂转**。
